# User Stories (MVP)

1. As a player, I want to see a ready puzzle when I open the page, so I can start immediately.
2. As a player, I want to upload my own image, so I can solve my personal photo.
3. As a player, I want to click a tile next to the empty cell to move it.
4. As a player, I want to move a tile only when it is adjacent to the empty cell.
5. As a player, I want to see a Victory message when the puzzle is solved.
6. As a player, I want to upload another image to start a new game.

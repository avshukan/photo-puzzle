# MVP Scope

## Included

- 15 puzzle logic.
- Fixed grid size: 4×4.
- One empty cell.
- Click on neighbor tile to move.
- Default image on first load.
- User can upload local image.
- Game restarts automatically after upload.
- Image is sliced into 16 parts using div + background-position.
- Start position is always solvable.
- Victory message is shown when solved.
- Minimalistic UI.

## Not included

- Show solution / auto solve.
- Timer.
- Move counter.
- Reset button.
- Drag & drop.
- Animations.
- Sounds.
- Server, accounts, persistence.
